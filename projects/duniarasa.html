Dunia Rasa

Culinary Exploration Website

Dunia Rasa is a web-based platform created to explore and celebrate the diversity of culinary experiences. It allows users to discover new dishes, learn about ingredients, and share their own creations. I developed this project using HTML, CSS, and JavaScript, focusing on a vibrant interface, intuitive navigation, and responsive layout that enhances user engagement.
